# Cloud Services

Cloud services designed for VPS deployment with full analysis capabilities and internet connectivity.

## Services

- **api-gateway**: Central API gateway and management interface
- **packet-analyzer**: Deep packet analysis and forensics
- **threat-intel**: Threat intelligence integration and analysis
- **vulnerability-scanner**: Vulnerability assessment and scanning
- **rule-generator**: Dynamic rule generation and updates

## Deployment

```bash
# Deploy cloud services
./deploy.sh vps
```

## Resource Requirements

- Memory: 2GB minimum
- Storage: 10GB minimum  
- Network: 1Gbps recommended
- Internet: Required for threat intel and scanning
